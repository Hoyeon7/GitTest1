HTML5에서 추가된 input 속성

입력창에 미리 글자를 적어놓게 세팅하는 기능
autofocus는 F5 새로고침시 입력창에 입력 준비 상태로 만들어주는 기능
list속성중 option입니다. 입력창에 메뉴 선택을 할수 있게 하는 기능.

multiple을 사용하면 여러 파일을 선택할수 있게 됨.

pattern은 들어가는 문자의 반복, 규칙
placeholder는 입력창에 회색빛으로 글자를 미리 나타나게 하는 기능
required는 placeholder 입력창에 글자를 입력하고 엔터칠경우 이곳 입력창에서 "이 입력란을 작성하세요" 라는 문구를 띄워줌
빈값일 경우 전달되지 않는다.
step은 화살표를 눌려서 숫자를 ?칸씩 건너뛰며 선택하게 하는 기능

버튼 기능을 원하는 이미지로 바꾸는 기능 버튼을 눌렸을때 다른 html화면으로 넘어가게 하는 기능.
그리고 인코딩을 바꾸거나 서버측 파일을 바꿀수 있다.
